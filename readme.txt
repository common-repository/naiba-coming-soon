A simple and effective coming soon plugin for WordPress.

== Description ==

Naiba Coming Soon is a lightweight plugin that displays a customizable coming soon page for your WordPress website. The plugin is easy to set up and provides a user-friendly interface for modifying the page title and content.

Main features:

* Easy to set up and configure
* Customizable page title and content
* Designed to work with any theme
* Secure and follows WordPress coding standards

== Installation ==

1. Upload the 'naiba-coming-soon' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Coming Soon to customize the page title and content.

== Frequently Asked Questions ==

= Can I use this plugin with any theme? =

Yes, the Naiba Coming Soon plugin is designed to work with any WordPress theme.

= How do I disable the coming soon page? =

To disable the coming soon page, simply deactivate the plugin through the 'Plugins' menu in WordPress.
